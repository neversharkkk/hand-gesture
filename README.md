# 手势识别与鼠标控制应用

这是一个使用 Python、OpenCV 和 MediaPipe 实现的实时手势识别与鼠标控制应用程序。

## 功能

- **实时手部跟踪**：使用 MediaPipe 实时检测并跟踪手部。
- **手势识别**：识别多种手势，包括握拳、手掌、剪刀、数字手势等。
- **鼠标控制**：通过手势控制鼠标移动和点击。
- **友好的用户界面**：包含标题栏、控制栏、状态显示和快捷键说明。
- **模式切换**：支持在手势识别模式和鼠标控制模式之间切换。
- **实时帧率显示**：显示当前处理帧率。

## 安装

确保你已经安装了 Python 3.7 或更高版本。然后安装所需的依赖库：

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install opencv-python mediapipe pyautogui
```

## 运行

运行主程序：

```bash
python main.py
```

## 操作说明

### 基本操作
- 程序启动后会自动打开摄像头。
- 按键盘上的 **'q'** 键退出程序。

### 模式切换
- 按 **'1'** 键切换到鼠标控制模式。
- 按 **'2'** 键切换到手势识别模式。

### 鼠标控制模式
- **移动鼠标**：伸直食指，移动手部控制鼠标光标。
- **点击**：同时伸直拇指和食指，将它们靠近执行点击操作。

## 技术说明

- **MediaPipe Hands**：用于手部关键点检测和跟踪。
- **OpenCV**：用于图像处理和显示。
- **PyAutoGUI**：用于鼠标控制。
- **性能优化**：使用图像缩放和参数调整提高处理速度。

## 系统要求

- Python 3.7 或更高版本
- 摄像头（内置或外置）
- Windows、macOS 或 Linux 操作系统
- 至少 2GB RAM

## 故障排除

### 摄像头无法打开
- 确保摄像头已连接并正常工作。
- 确保没有其他程序正在使用摄像头。
- 尝试使用不同的摄像头索引（修改 `cv2.VideoCapture(0)` 中的参数）。

### 手势识别不准确
- 确保手部在摄像头视野中清晰可见。
- 确保光线充足，避免手部阴影。
- 尝试调整摄像头角度，使手部保持在画面中央。

### 鼠标控制不流畅
- 确保手部移动平稳。
- 尝试调整鼠标灵敏度（修改代码中的 `mouse_sensitivity` 参数）。
- 确保电脑性能足够，避免同时运行过多程序。
